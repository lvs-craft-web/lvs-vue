  1. Задача 6 =>

    Пари для танців. Поступово вибираємо хлопця, дівчину i додаємо у обрані пари. Пару можна видалити. Поки не вибрано хлопця i дівчину кнопка «Додати» заблокована. Якщо не вистачає хлопців або дівчат вибір також блокується.

  2. Задача 8 =>

    Дано список автомобілів (марка, рік випуску, ціна). Сформувати елементи для фільтрування з використанням випадаючого списку (контент цих випадаючих списків сформувати у залежності від переданого списку).

  3. Задача 9 =>

    Перекладач. Користувачу виводять змішані картки з словами на англійській i українській мові. Користувач поступово клікає на картки (виділяємо синьою рамкою). Якщо знайдено правильні пари карток, що відповідають одному слову, то видаляємо ці картки. Інакше - виділяємо червоною рамкою i через секунду забираємо рамку.

  4. Задача 12 =>

    Мережа магазинів. Дано список відділень та список товарів. Для кожного відділення можна вибирати декілька товарів. Вибирати та відображати перелік вибраних товарів для кожного відділення.
